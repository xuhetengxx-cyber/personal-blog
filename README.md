# personal-blog

一个极简的个人博客网页骨架，使用原生 HTML、CSS、JavaScript 搭建，方便后续快速改造成你的个人主页或博客。

## 当前包含

- 首页 Hero 区块
- 关于我模块
- 最新文章卡片占位
- 联系方式模块
- 简洁深色主题样式

## 本地预览

直接双击 `index.html` 打开，或运行：

```bash
python3 -m http.server 8000
```

然后访问：`http://localhost:8000`

1. 把文案替换成你的真实信息

3. 配置 GitHub Pages 自动发布
